# 🛡️ Oh-My-Security (OMS)

> 超级个体（独立开发者/创作者）的数字护卫。

**Oh-My-Security** 是一款专为**个人开发者、Indie Hackers 和自媒体创作者**打造的轻量级终端安全工具。

我们不谈枯燥的企业级 DevSecOps 和等保合规，我们只关心关系到超级个体"身家性命"的切身痛点：
**如何防止不小心把 API Key 传到 GitHub 导致破产？如何防止刚写好的文章因为包含敏感词被平台封号？如何防止自己刚上线的小工具站被羊毛党刷爆服务器？**

OMS 将顶级安全专家的知识库和 AI 审计能力封装进一个极简的命令行中，让你在专注创造的同时，免受数字世界的暗箭。

## ✨ 核心场景

### 1. 💻 个人代码防破产审计 (`oms code`)
独立开发者往往缺乏代码 Review 环境。在 `git push` 之前跑一次 OMS：
- 深度扫描硬编码的 Secret、数据库密码、各种大模型 API Key（防破产保护）。
- 识别个人全栈项目中常犯的低级鉴权越权漏洞、SQL 注入风险。

### 2. 📝 创作者内容风控排雷 (`oms content`)
辛苦写的深度文章，发出去却被平台无情封禁？
- 发布前扫描 Markdown/文本草稿，识别可能触发各大平台封控的敏感词汇、争议性暴论。
- 检查文章配图或文本中是否不小心泄露了真实的手机号、身份证、家庭住址等隐私数据。

### 3. 🛡️ 独立站防白嫖指南 (`oms bizsec`)
个人做的小工具站/小程序刚有点流量，接口就被黑灰产的爬虫狂刷？
- 结合内置的 100+ 真实业务安全案例，快速诊断你的公开 API 是否裸奔。
- 提供低成本、好上手的极简防刷代码片段（如：基于 IP 的令牌桶限流、简单指纹校验）。

### 4. ☁️ 廉价云主机一键加固 (`oms vps`)
刚买了一台几十块钱的轻量云主机搭博客，不知道怎么防黑客？
- 提供"傻瓜式"的服务器加固向导，帮你一键生成 SSH 密钥配置、防火墙规则和防暴力破解脚本。

## 🚀 快速开始

### 安装 OMS
克隆本仓库，并将 `bin` 目录添加到您的环境变量中：

```bash
git clone https://github.com/tinylion1024/oh-my-security.git
cd oh-my-security
pip install -r requirements.txt

# 将 oms 加入全局命令 (建议写入 ~/.zshrc 或 ~/.bashrc)
export PYTHONPATH=$(pwd)
export PATH="$PATH:$(pwd)/bin"
```

### 唤起你的安全护卫

在终端中输入 `oms`，获取你所需的防护：

#### 查询防坑案例
例如，想了解怎么防止自己的网站被别人自动化抓取：
```bash
oms case "爬虫"
```

#### 漏洞快速定级
发现一个第三方库有漏洞，想快速知道严不严重：
```bash
oms cvss -av N -ac L -pr N -ui N -c H -i H -a H
```

## 🧠 核心资产：300+ 避坑案例库

OMS 并不是空壳工具，它的智慧来源于内置的 `knowledge_base` 目录下高达 **304 个** 深度解析的实战避坑案例。
这些不仅是大模型进行 AI 审计时的重要上下文（Context），也是你平时阅读学习的极佳素材。

- **业务风控 (Bizsec)**：100个案例，教你识别薅羊毛、接口滥用、0元下单陷阱。
- **信息安全 (Infosec)**：100个案例，带你避开配置失误、供应链投毒和基础设施漏洞。
- **AI 安全 (AIsec)**：100个案例，为你接入大模型应用时防止 Prompt 注入、数据泄露提供指引。
