# 隐私政策缺失风险（Privacy Policy Missing）

## 元数据
- **tier适用**: L1 ✅ | L2 ✅ | L3 ✅
- **领域**: compliance（合规自查）
- **严重程度**: critical
- **修复成本**: L1: 免费 | L2: $100-500 | L3: 法律顾问
- **独立开发者适用度**: ⭐⭐⭐⭐⭐ (5/5)
- **最后验证日期**: 2026-06-13

---

## L1 独立开发者版（速查版）

### 你会遭遇什么（一句话）
未提供隐私政策，违反多项数据保护法规，面临罚款和下架。

### 一分钟识别
你的产品是否有以下特征：
- [ ] 无隐私政策页面
- [ ] 隐私政策难以找到
- [ ] 内容不完整
- [ ] 无中文版本
- [ ] 长期未更新
→ 勾选≥2项，即需立即关注

### 一句话防御
制定完整隐私政策，放在显著位置，定期更新。

### 快速行动清单
1. **立即行动**（今天可完成，免费）
   - [ ] 使用模板创建隐私政策
   - [ ] 放在网站底部或设置页面
   - [ ] 包含必要内容

2. **短期行动**（本周可完成，低成本）
   - [ ] 请律师审核
   - [ ] 翻译成用户语言
   - [ ] 添加版本和更新日期

3. **长期行动**（规划中）
   - [ ] 定期审查更新
   - [ ] 记录用户同意

## L2 小团队版（理解版）

### 隐私政策必要内容

```
必须包含:
1. 数据控制者信息
2. 收集的数据类型
3. 数据使用目的
4. 法律依据
5. 数据接收方
6. 跨境传输信息
7. 数据保留期限
8. 用户权利说明
9. Cookie 使用说明
10. 联系方式
```

### 隐私政策模板

```markdown
# 隐私政策

**最后更新日期**: YYYY-MM-DD

## 1. 我们是谁
[公司名称] 是本网站/应用的数据控制者。

联系方式: [邮箱/地址]

## 2. 我们收集哪些数据
- 账户信息: 用户名、邮箱
- 使用数据: 访问记录、设备信息
- [其他数据]

## 3. 我们如何使用数据
- 提供、维护服务
- 改进用户体验
- [其他用途]

## 4. 法律依据
- 合同履行
- 用户同意
- 合法利益

## 5. 数据共享
我们可能与以下方共享数据:
- 服务提供商
- [其他]

## 6. 数据保留
[保留期限说明]

## 7. 您的权利
- 访问权
- 更正权
- 删除权
- [其他权利]

## 8. Cookie 使用
[Cookie 说明]

## 9. 联系我们
[联系方式]
```

### 检测清单

- [ ] 隐私政策存在
- [ ] 内容完整
- [ ] 易于访问
- [ ] 有更新日期
- [ ] 用户语言版本
