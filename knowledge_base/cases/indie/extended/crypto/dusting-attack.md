## L1 独立开发者版（速查版）

### 你会遭遇什么（一句话）
收到极小额代币，攻击者追踪你的地址关联和交易行为。

### 一分钟识别
钱包是否有以下特征：
- [ ] 收到未知小额代币
- [ ] 有 0.0001 级别的转账
- [ ] 收到不认识的 NFT
- [ ] 有不明的空投
→ 有则可能被粉尘攻击

### 一句话防御
不要触碰未知代币，忽略或使用隐私解决方案。

### 快速行动清单
1. **立即行动**（今天可完成，免费）
   - [ ] 检查钱包未知代币
   - [ ] 不要转移或出售它们
   - [ ] 了解粉尘攻击原理

2. **短期行动**（本周可完成，免费）
   - [ ] 考虑使用隐私工具
   - [ ] 分离交易地址
   - [ ] 隐藏小额资产

3. **长期行动**（规划中）
   - [ ] 使用新地址接收敏感交易
   - [ ] 定期更换地址

## L2 小团队版（理解版）

### 粉尘攻击原理

```
攻击目的:
1. 地址关联: 追踪多个地址属于同一人
2. 行为分析: 分析交易模式和习惯
3. 入口寻找: 为后续攻击寻找入口
4. 诱导交互: 引导用户与恶意合约交互
```

### 防护措施

```javascript
// 防粉尘策略
const dustingProtection = {
  // 处理未知代币
  unknownTokens: {
    doNotTransfer: true,          // 不转移
    doNotSell: true,              // 不出售
    hideFromView: true,           // 隐藏显示
    doNotInteract: true           // 不交互
  },

  // 隐私保护
  privacy: {
    useNewAddress: true,          // 新交易用新地址
    separateWallets: true,        // 分离钱包用途
    coinControl: true             // 使用币控制
  }
};
```
