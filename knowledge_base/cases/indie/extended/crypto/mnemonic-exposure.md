## L1 独立开发者版（速查版）

### 你会遭遇什么（一句话）
12/24 个助记词泄露，所有关联资产被转移，无法恢复。

### 一分钟识别
你的助记词管理是否有以下风险：
- [ ] 数字形式存储
- [ ] 单一备份
- [ ] 在网络上传输过
- [ ] 告诉过他人
- [ ] 备份位置不安全
→ 勾选≥2项，即需立即关注此风险

### 一句话防御
手写备份，多点存储，永不联网传输。

### 快速行动清单
1. **立即行动**（今天可完成，免费）
   - [ ] 检查助记词存储方式
   - [ ] 创建手写备份
   - [ ] 销毁数字记录

2. **短期行动**（本周可完成，中成本）
   - [ ] 购买金属备份板
   - [ ] 创建分散备份
   - [ ] 转移资产到新钱包

3. **长期行动**（规划中）
   - [ ] 建立安全备份策略
   - [ ] 定期检查备份完整性

### 推荐工具
- **免费**：
  - 纸和笔
  - 安全存放盒

- **低成本**：
  - 金属助记词板 ($30-100)
  - 防火防水保险箱

### 验证方法
- [ ] 测试步骤1：验证备份可读性
- [ ] 测试步骤2：检查备份位置安全
- [ ] 测试步骤3：确认无数字记录

## L2 小团队版（理解版）

### 助记词风险场景

```
泄露途径:
1. 截图保存
2. 云笔记记录
3. 邮件发送
4. 聊天软件传输
5. 钓鱼网站输入
6. 假钱包记录
7. 物理窃取
8. 偷拍记录
```

### 安全存储策略

```javascript
// 助记词安全配置
const mnemonicSecurity = {
  // 存储介质
  storage: {
    format: 'handwritten',        // 手写
    material: 'metal',            // 金属板
    copies: 3,                    // 3份备份
    locations: ['home_safe', 'bank', 'family']
  },

  // 分割存储（可选）
  shamirSharing: {
    enabled: true,
    shares: 5,                    // 5份
    threshold: 3,                 // 需3份恢复
    locations: ['safe', 'bank', 'family_a', 'family_b', 'lawyer']
  },

  // 严禁行为
  prohibited: {
    digitalStorage: true,         // 禁止数字存储
    networkTransmission: true,    // 禁止网络传输
    screenshot: true,             // 禁止截图
    shareWithOthers: true         // 禁止分享
  }
};
```
