## L1 独立开发者版（速查版）

### 你会遭遇什么（一句话）
私钥或助记词泄露，攻击者瞬间转移你所有加密资产，无法追回。

### 一分钟识别
你的私钥存储是否有以下风险：
- [ ] 存储在联网设备
- [ ] 截图或拍照保存
- [ ] 发送到云端或邮件
- [ ] 告诉过他人
- [ ] 只有一份备份
→ 勾选≥2项，即需立即关注此风险

### 一句话防御
私钥永不联网，使用硬件钱包，多点物理备份。

### 快速行动清单
1. **立即行动**（今天可完成，免费）
   - [ ] 检查私钥存储方式
   - [ ] 创建安全的离线备份
   - [ ] 清理数字存储的私钥

2. **短期行动**（本周可完成，中成本）
   - [ ] 购买硬件钱包
   - [ ] 转移资产到硬件钱包
   - [ ] 销毁旧备份

3. **长期行动**（规划中）
   - [ ] 建立多重备份策略
   - [ ] 定期安全审查

### 推荐工具
- **免费**：
  - 纸钱包（手写）
  - 金属助记词板

- **低成本**：
  - Ledger Nano ($59+)
  - Trezor ($69+)

### 验证方法
- [ ] 测试步骤1：验证硬件钱包功能
- [ ] 测试步骤2：检查备份安全性
- [ ] 测试步骤3：确认私钥不在联网设备

## L2 小团队版（理解版）

### 私钥泄露途径

```
泄露途径:
1. 钱包软件漏洞
2. 恶意软件窃取
3. 剪贴板监控
4. 云存储泄露
5. 截图自动上传
6. 钓鱼网站输入
7. 社会工程攻击
8. 物理设备被盗
```

### 安全存储策略

```javascript
// 私钥安全存储配置
const privateKeySecurity = {
  // 存储介质
  storage: {
    primary: 'hardware_wallet',   // 硬件钱包
    backup: 'metal_plate',        // 金属备份
    locations: 2,                 // 2个物理位置
    encryption: 'offline'         // 离线加密
  },

  // 备份策略
  backup: {
    method: 'shamir',             // Shamir 分割
    shares: 3,                    // 3份
    threshold: 2,                 // 需2份恢复
    locations: ['safe', 'bank', 'trusted_family']
  },

  // 使用规则
  usage: {
    neverOnline: true,            // 永不联网
    neverPhoto: true,             // 不拍照
    neverCloud: true,             // 不上云
    neverShare: true              // 不分享
  }
};
```

### 检测清单

- [ ] 私钥不在联网设备
- [ ] 有多个物理备份
- [ ] 备份存储安全
- [ ] 使用硬件钱包
- [ ] 无数字记录
