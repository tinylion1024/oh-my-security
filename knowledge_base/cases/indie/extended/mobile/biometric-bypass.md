## L1 独立开发者版（速查版）

### 你会遭遇什么（一句话）
攻击者通过照片、3D 打印、强制手段等方式绕过生物识别锁，获取设备访问权限。

### 一分钟识别
你的生物识别设置是否有以下特征：
- [ ] 仅使用生物识别无备用密码
- [ ] 生物识别超时时间过长
- [ ] 未启用生物识别失败降级
- [ ] 敏感操作仅靠生物识别
- [ ] 未了解生物识别法律风险
→ 勾选≥2项，即需立即关注此风险

### 一句话防御
设置强密码作为备用，缩短生物识别超时时间，关键操作需要密码确认。

### 快速行动清单

## L2 小团队版（理解版）

### 绕过方式分析

| 方式 | 适用对象 | 成功率 | 防护措施 |
|------|---------|--------|---------|
| 照片欺骗 | 人脸识别（低端） | 高 | 使用红外/3D识别 |
| 3D 面具 | 人脸识别（部分） | 中 | 活体检测 |
| 指纹复刻 | 指纹识别 | 中 | 电容传感器 |
| 强制解锁 | 所有生物识别 | 高 | 必须密码选项 |
| 法律强制 | 所有生物识别 | 高 | 隐私密码 |

### 安全配置

```javascript
// 生物识别安全策略
const biometricPolicy = {
  // 设备解锁
  deviceUnlock: {
    biometricAllowed: true,
    fallbackToPassword: true,
    timeout: 5,  // 5分钟后需密码
    failedAttempts: 3  // 3次失败后需密码
  },

  // 敏感操作
  sensitiveActions: {
    payment: 'require_password',      // 支付必须密码
    password_change: 'require_password',  // 修改密码必须密码
    data_export: 'require_password',  // 导出数据必须密码
    login_notification: true          // 登录通知
  },

  // 应用内认证
  appAuth: {
    biometricTimeout: 5 * 60 * 1000,  // 5分钟
    backgroundTimeout: 30 * 1000,     // 后台30秒需重新认证
    sensitiveActionPassword: true     // 敏感操作需密码
  }
};
```

### 检测清单

- [ ] 设置了强备用密码
- [ ] 生物识别超时合理（≤5分钟）
- [ ] 多次失败后降级到密码
- [ ] 敏感操作需要密码
- [ ] 了解法律风险
- [ ] 敏感应用有独立保护
