# 结算欺诈

## 元数据

| 字段 | 值 |
|------|-----|
| 编号 | PAY-009 |
| 领域 | 支付安全 |
| 适用层级 | L1 ✅ |
| 创建日期 | 2026-06-11 |

---

## 一句话风险

攻击者篡改结算账户信息或利用对账漏洞，将商家资金转移至非法账户。

---

## 一分钟识别清单

- [ ] **账户变更验证**：结算账户变更是否需要双重确认（邮箱、短信、人工审核）？
- [ ] **对账一致性**：订单金额、支付金额与结算金额是否三方对账一致？
- [ ] **异常结算监控**：是否存在结算账户频繁变更或大额结算到新账户？

---

## 一句话防御

结算账户变更实施多因素验证和冷静期，建立自动对账系统，异常结算触发人工审核。
