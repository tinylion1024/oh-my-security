## L1 独立开发者版（速查版）

### 你会遭遇什么（一句话）
收到伪装成官方、合作方或朋友的私信，诱导点击链接或提供敏感信息。

### 一分钟识别
私信是否有以下特征：
- [ ] 要求点击链接
- [ ] 紧急语气催促
- [ ] 承诺投资机会
- [ ] 中奖或赠品通知
- [ ] 要求提供账号信息
→ 勾选≥2项，可能为钓鱼私信

### 一句话防御
不点击私信中的链接，通过官方渠道核实，举报可疑私信。

## L2 小团队版（理解版）

### 私信钓鱼类型

| 类型 | 话术 | 危害 |
|------|------|------|
| 投资诈骗 | 高收益投资项目 | 财产损失 |
| 官方冒充 | 账号异常需验证 | 账号被盗 |
| 中奖诈骗 | 恭喜中奖领奖 | 信息泄露 |
| 合作诈骗 | 商务合作机会 | 欺诈损失 |
| 求助诈骗 | 朋友急需用钱 | 财产损失 |

### 防护措施

```javascript
// 私信安全配置
const dmSecurity = {
  // 接收设置
  receiving: {
    allowFrom: 'following',       // 仅关注的人
    filterSpam: true,             // 过滤垃圾
    filterKeywords: ['中奖', '投资', '验证']
  },

  // 自动过滤
  autoFilter: {
    links: 'warn',                // 链接警告
    attachments: 'block',         // 阻止附件
    suspicious: 'report'          // 自动举报
  },

  // 用户教育
  education: {
    showWarning: true,            // 显示警告
    verifyOfficial: true          // 验证官方账号
  }
};
```
