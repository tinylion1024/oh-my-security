## L1 独立开发者版（速查版）

### 你会遭遇什么（一句话）
发布的快拍/动态被截屏保存、恶意转发或泄露隐私信息。

### 一分钟识别
你的快拍设置是否有以下特征：
- [ ] 所有人可见
- [ ] 允许转发分享
- [ ] 未隐藏好友列表
- [ ] 显示位置信息
- [ ] 背景暴露隐私
→ 勾选≥2项，即需立即关注此风险

### 一句话防御
设置快拍可见范围，禁用转发，注意背景隐私。

## L2 小团队版（理解版）

### 快拍风险类型

| 风险 | 说明 | 防护 |
|------|------|------|
| 截屏保存 | 他人截屏保存 | 限制可见范围 |
| 第三方工具 | 使用工具保存 | 无法防护 |
| 转发泄露 | 被转发到其他平台 | 禁用转发 |
| 背景隐私 | 照片背景暴露信息 | 注意背景 |
| 位置追踪 | 位置信息被利用 | 隐藏位置 |

### 安全配置

```javascript
// 快拍安全设置
const storySecurity = {
  // 可见性
  visibility: {
    audience: 'close_friends',    // 密友可见
    hideFrom: [],                 // 隐藏特定用户
    blockSharing: true            // 禁止分享
  },

  // 互动限制
  interactions: {
    allowReplies: 'following',    // 仅关注者可回复
    allowMentions: false,         // 禁止提及
    allowLinks: false             // 禁止链接
  },

  // 隐私保护
  privacy: {
    hideLocation: true,           // 隐藏位置
    hideTags: true,               // 隐藏标签
    autoDelete: 24                // 24小时自动删除
  }
};
```
