# 黑客利用大模型定制化生成高隐蔽性钓鱼邮件与社工脚本
- **所属领域**: AI 安全
- **威胁类型**: 社会工程学攻击 / 大模型滥用
- **事件回顾**: 某大型企业遭受了一次极具针对性的 APT 攻击，导致核心研发数据泄露。攻击的突破口在于一系列高度定制化的鱼叉式钓鱼邮件。不同于以往语法生硬、内容模板化的钓鱼邮件，此次攻击者发送的邮件行文极其自然，不仅完美模仿了企业内部高管和外部合作伙伴的写作风格，还精准切中了受害员工近期的工作项目和心理弱点。员工在毫无防备的情况下点击了包含恶意负载的链接，致使内网被突破。
- **Red View (利用路径)**: 攻击者首先通过 OSINT (开源情报) 收集目标企业的高管言论、内网公开文档以及目标员工的社交动态。然后，利用如 WormGPT 或未经审查的开源大模型，将这些情报作为 Prompt 输入，要求模型生成符合特定人物口吻、特定业务场景的钓鱼邮件和后续的聊天回复脚本。大模型的上下文理解和自然语言生成能力，使得钓鱼攻击的成本大幅降低，且迷惑性呈指数级上升。
- **Blue View (防御缺失)**: 传统的邮件安全网关 (SEG) 主要依赖发件人信誉、域名签名 (SPF/DKIM/DMARC) 以及已知恶意链接和附件哈希进行拦截，对于语义合法、语境真实且未使用已知恶意域名的 AIGC 钓鱼邮件束手无策。企业内部员工的安全意识培训内容滞后，未能涵盖 AI 赋能的新型社工攻击手段。
- **策略借鉴**: 升级邮件安全防护体系，引入自然语言理解 (NLU) 技术进行邮件行为意图和语义异常分析，识别语言风格的突变或异常的紧急性诱导；加强零信任架构落地，即使员工终端被控，也能通过严格的身份和设备持续验证阻止横向移动；更新安全意识培训，加入 AIGC 钓鱼邮件的识别演练，建立更敏捷的内部异常报告机制。
