# 陌生人社交平台“杀猪盘”自动化话术渗透攻击
- **所属领域**: 业务安全
- **威胁类型**: 诈骗风险 (Fraud)
- **事件回顾**: 某海外社交软件遭遇高度自动化的诈骗团伙攻击。攻击者通过模拟器脚本批量注册女性画像账号，并利用 AI 大模型生成的实时话术与男性用户互动。在成功建立信任后，通过自动化消息推送诱导用户下载博彩应用。该事件导致该平台月活跃用户举报率上升 40%，并在多个地区应用商店被警告下架。
- **Red View (利用路径)**: 
    1. **批量注册**: 利用海外短信接码平台（如 SMS-Activate）通过注册接口获取大量账号。
    2. **画像自动化**: 使用 Python 脚本结合 AI 绘图（Midjourney）生成大量高吸引力的虚拟肖像，并自动完善资料。
    3. **大模型集成**: 通过 API 接入 GPT-4 接口，根据对方的回复内容实时生成富有情感的回复话术，绕过平台的静态关键词过滤。
    4. **引流跳转**: 话术中巧妙植入经过混淆的短链接（Short URL），逃避 WAF 对敏感域名的拦截。
- **Blue View (防御缺失)**: 
    1. **文本审核滞后**: 基于黑产关键词的正则表达式无法识别 AI 生成的自然语言，语义层面的诈骗识别能力不足。
    2. **群发检测失效**: 攻击者通过控制回复频率，使自动化行为看起来更像真人，绕过了基于“发信频率”的朴素防御。
    3. **设备指纹误报**: 团伙使用了高性能真机农场，平台无法通过传统的模拟器特征检测将其剔除。
- **策略借鉴**: 
    1. **NLP 语义建模**: 引入基于 Transformer 的语义异常检测模型，识别话术中常见的“情感铺垫-金钱诱导”这一特定的诈骗意图链。
    2. **用户投诉闭环**: 建立投诉权重系统，对于高举报率的账号实施实时灰度限制，人工介入二次审核。
    3. **链接信誉分级**: 强制对所有外链进行沙箱重定向和信誉评分，对低信誉域名的跳转实施风险弹窗提醒。
