# 域名资产安全检查清单

> 10分钟快速检查你的域名资产是否存在致命风险

---

## 🔴 必须检查 (Critical)

### 域名注册
- [ ] **域名锁定**: 是否开启域名锁定/转移锁？
- [ ] **WHOIS隐私**: 是否开启 WHOIS 隐私保护？
- [ ] **邮箱安全**: 注册邮箱是否开启 2FA？
- [ ] **联系信息**: 联系邮箱是否有效？

### SSL证书
- [ ] **证书有效期**: SSL证书是否在有效期内？
- [ ] **证书链**: 证书链是否完整？
- [ ] **协议版本**: 是否禁用 TLS 1.0/1.1？
- [ ] **证书透明**: 是否监控证书透明度日志？

### DNS安全
- [ ] **DNSSEC**: 是否启用 DNSSEC？
- [ ] **SPF记录**: 是否配置 SPF 记录？
- [ ] **DKIM记录**: 是否配置 DKIM 记录？
- [ ] **DMARC记录**: 是否配置 DMARC 记录？

---

## 🟠 重要检查 (High)

### 域名监控
- [ ] **到期提醒**: 是否设置域名到期提醒？
- [ ] **多渠道提醒**: 是否设置邮件+短信提醒？
- [ ] **自动续费**: 是否开启自动续费？
- [ ] **备份支付**: 是否有备份支付方式？

### 子域名管理
- [ ] **子域名审计**: 是否定期审计子域名？
- [ ] **废弃子域名**: 是否清理废弃子域名？
- [ ] **子域名证书**: 所有子域名是否都有证书？
- [ ] **DNS解析**: 废弃DNS记录是否清理？

---

## 🟡 建议检查 (Medium)

### DNS配置
- [ ] **CAA记录**: 是否配置 CAA 记录？
- [ ] **TTL设置**: TTL是否设置合理？
- [ ] **DNS提供商**: 是否使用可靠的DNS提供商？
- [ ] **DNS备份**: 是否有DNS配置备份？

---

## 📋 快速自测命令

```bash
# 检查域名安全
oms domain check example.com

# 检查SSL证书
oms domain ssl example.com

# 检查DNS配置
oms domain dns example.com

# 查看最佳实践
oms domain guide
```

---

> 记住：**域名是你的数字身份，一次劫持可能让你永远失去它。**
