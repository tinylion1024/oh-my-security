# 内部威胁

## 严重程度
🟠 高

## 识别信号
- 员工异常访问敏感数据
- 离职前大量数据下载
- 异常的权限提升请求
- 未授权的系统访问
- 数据外泄迹象
- 可疑的远程访问
- 异常的邮件转发规则
- 安全策略被绕过

## 立即行动 (0-15分钟)

1. **确认威胁**
   - 验证可疑行为
   - 确认涉及人员
   - 评估威胁等级
   - 记录发现时间

2. **控制措施**
   - 暂停涉事员工权限
   - 回收公司设备
   - 限制系统访问
   - 保护关键数据

3. **通知升级**
   - 通知安全团队
   - 告知 HR 部门
   - 通知法务部门
   - 准备管理层汇报

4. **证据保全**
   - 保存访问日志
   - 截取操作记录
   - 保留通信记录
   - 记录异常行为时间线

## 短期处理 (1-4小时)

1. **调查分析**
   - 审计员工系统活动日志
   - 分析数据访问记录
   - 确定数据泄露范围
   - 追踪异常行为时间线

2. **动机分析**
   - 评估是否恶意意图
   - 确定是否利益驱动
   - 分析是否存在不满情绪
   - 评估是否外部胁迫

3. **影响评估**
   - 评估数据损失/泄露
   - 确定业务影响
   - 评估知识产权损失
   - 计算潜在损失

4. **决策制定**
   - 确定处理方式（内部/法律）
   - 准备证据材料
   - 制定沟通策略
   - 安排人员处理

## 恢复步骤

1. **权限清理**
   - 撤销所有访问权限
   - 禁用所有账户
   - 更改共享密码
   - 收回访问令牌

2. **数据恢复**
   - 恢复被删除/篡改的数据
   - 评估数据泄露影响
   - 必要时通知受影响方
   - 处理法律合规要求

3. **系统加固**
   - 实施最小权限原则
   - 加强访问审计
   - 部署用户行为分析 (UBA)
   - 加强离职审计流程

4. **组织改进**
   - 完善背景调查流程
   - 加强安全意识培训
   - 建立举报机制
   - 改善员工关系
