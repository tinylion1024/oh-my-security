# 社会工程

## 严重程度
🟡 中

## 识别信号
- 用户报告可疑邮件/电话
- 钓鱼网站被发现
- 异常的登录行为
- 用户信息被泄露的报告
- 供应商/合作伙伴异常请求
- 可疑的身份验证请求
- 员工报告被诱导操作
- 社交媒体出现假冒账号

## 立即行动 (0-15分钟)

1. **确认攻击**
   - 验证攻击类型（钓鱼/假冒/诈骗）
   - 确认攻击目标
   - 评估攻击范围
   - 记录发现时间

2. **阻止攻击**
   - 下架钓鱼网站（联系域名商/服务商）
   - 封禁攻击者邮箱/账号
   - 阻止可疑链接
   - 阻断正在进行的攻击

3. **通知相关方**
   - 通知安全团队
   - 告知潜在受害者
   - 通知品牌保护团队
   - 启动事件记录

4. **证据保全**
   - 截取钓鱼网站/邮件
   - 保存攻击者信息
   - 记录攻击手法
   - 收集受害者报告

## 短期处理 (1-4小时)

1. **攻击调查**
   - 分析攻击手法和模板
   - 追踪攻击来源
   - 确定受影响人数
   - 评估信息泄露情况

2. **影响评估**
   - 确定泄露的凭据/信息
   - 评估潜在风险
   - 识别受害账户
   - 确定业务影响

3. **受害救助**
   - 重置可能泄露的密码
   - 撤销可能泄露的令牌
   - 取消可能泄露的凭证
   - 通知受影响用户

4. **防护加强**
   - 加强邮件安全检测
   - 部署反钓鱼方案
   - 更新钓鱼签名库
   - 完善身份验证流程

## 恢复步骤

1. **清理残留**
   - 移除所有钓鱼内容
   - 清理假冒账号
   - 删除钓鱼邮件
   - 清理恶意链接

2. **账户保护**
   - 为受害账户启用 MFA
   - 重置相关密码
   - 检查账户异常活动
   - 加强账户监控

3. **用户教育**
   - 发送安全提醒
   - 分享攻击案例分析
   - 提供识别钓鱼指南
   - 建立报告渠道

4. **防护升级**
   - 完善邮件网关过滤
   - 加强钓鱼检测能力
   - 部署品牌保护方案
   - 定期钓鱼演练
