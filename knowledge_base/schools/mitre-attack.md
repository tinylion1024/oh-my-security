# MITRE ATT&CK 框架 - 攻防对抗知识库

## 1. 核心定义
MITRE ATT&CK 是一个基于真实观察到的攻击行为而建立的全球可访问的对抗策略和技术库。

## 2. 核心维度 (Matrix)
- **Tactics (战术)**: 攻击者的目标（如：持久化、提权、横向移动）。
- **Techniques (技术)**: 实现目标的具体手段（如：利用特定的系统漏洞）。
- **Procedures (过程)**: 技术执行的特定序列。

## 3. 对决策的意义
- **红队**: 用于模拟真实攻击路径，确保覆盖所有可能的战术维度。
- **蓝队**: 用于映射现有的检测与防御能力，识别监控盲点。
- **治理**: 建立统一的攻防语言，将零散的漏洞映射为系统的威胁。
