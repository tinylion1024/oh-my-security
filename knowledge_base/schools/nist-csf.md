# NIST 网络安全框架 (NIST CSF)

## 1. 五大核心功能 (Core Functions)
1. **Identify (识别)**: 识别关键资产、风险及管理流程。
2. **Protect (保护)**: 实施必要的安全措施。
3. **Detect (检测)**: 及时发现安全事件。
4. **Respond (响应)**: 对检测到的事件采取行动。
5. **Recover (恢复)**: 恢复业务韧性。

## 2. 设计哲学
- **风险导向**: 而非清单导向。
- **持续改进**: 安全是一个动态循环。
- **通用语言**: 桥接技术团队与高层管理者的沟通鸿沟。
