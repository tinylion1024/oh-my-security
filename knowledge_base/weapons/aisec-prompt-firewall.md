# AI 提示词防火墙 (Prompt Firewall/Filter)

- **分类**: 预防 + 检测
- **适用场景**: 解决针对 LLM 的提示词注入 (Prompt Injection)、恶意指令执行、以及模型幻觉导致的违规输出。

## 工作原理
在用户输入进入大模型之前，通过专门的安全层进行语义扫描和拦截。

1. **关键词匹配**: 拦截已知的注入模板（如 "Ignore all previous instructions"）。
2. **语义分类器**: 使用轻量级分类模型 (如 BERT) 识别输入的意图是否包含对抗性。
3. **PII 脱敏**: 自动识别并掩盖用户输入中的身份证号、手机号等隐私信息。
4. **输出比对**: 将模型输出与输入进行比对，防止模型被诱导输出训练集中的敏感数据。

## 执行建议
- **响应延迟**: 安全层应极快，通常建议控制在 50ms 以内。
- **动态更新**: 攻击模板更新极快，需要具备云端同步威胁情报的能力。
- **推荐工具**: NeMo Guardrails (NVIDIA), Lakera Guard, Microsoft Azure AI Content Safety.
