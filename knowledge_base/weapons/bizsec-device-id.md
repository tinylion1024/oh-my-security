# 设备指纹 (Device Fingerprinting) SDK

- **分类**: 检测 + 预防
- **适用场景**: 解决账户撞库、多账号薅羊毛、模拟器作弊、以及垃圾注册问题。

## 工作原理
通过在移动端 (App) 或网页端 (Web) 集成 SDK，采集数百个非敏感的硬件和软件特征，通过算法生成全局唯一的设备 ID。

1. **特征采集**: Canvas 绘图特征、WebGL 渲染、字体列表、传感器噪点、硬件序列号等。
2. **风险识别**: 识别环境是否为模拟器、是否处于调试模式 (Debug)、是否有改机软件 (Hook 框架)。
3. **黑产库对比**: 将生成的设备 ID 与已知的黑产设备库进行碰撞。

## 执行建议
- **隐私合规**: 确保 SDK 采集的项符合 GDPR 和个保法要求，需在隐私协议中明确告知。
- **稳定性**: 必须具备抗清理（清空缓存、重装系统后依然能识别）的能力。
- **推荐工具**: 数美、同盾、数蓬、或自研基于 Open Source 方案。
